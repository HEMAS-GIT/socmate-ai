## AI Assistance

This project was developed with the assistance of OpenAI tools during the development process.

### How GPT-5.6 was used

* Brainstormed the project architecture and feature set.
* Assisted in designing the FastAPI backend and React frontend structure.
* Helped generate and refine API endpoints.
* Assisted with debugging deployment issues and improving code quality.
* Helped write project documentation, the README, and Devpost submission content.

### How Codex was used

* Assisted with code generation and refactoring.
* Helped implement FastAPI routes and backend logic.
* Suggested improvements to React components.
* Assisted in debugging integration issues between the frontend and backend.
* Helped optimize the project structure and development workflow.

All generated code was reviewed, modified, tested, and integrated by the team before being included in the final project.
